# 1 主快捷键

Ctrl + K + Ctrl + S （查看）键盘快捷方式

F1 或 Ctrl + Shift + P 主命令框（查找并运行指令）

Shift + P 主命令框

# 1.1 主命令框补充

主命令框输入 ? 可以查看所有提示

主要的几个符号：

  • > 用于显示所有命令
  • @ 用于显示和跳转文件中的“符号”（Symbols），在@符号后面添加 : 则可以把符号们按类别归类。
  • # 用于显示和跳转工作区中的“符号”（Symbols）
  • : 用于跳转到当前文件中的某一行

# 2 快捷键

# 2.1 左侧面板

Ctrl + Shift + E 文件

Ctrl + Shift + D 运行和调试

Ctrl + B 专注模式：隐藏左侧面板

Ctrl + J 专注模式：隐藏下方控制台面板

# 2.2 查看类

Ctrl + R 快捷切换窗口

Ctrl + F 查找

F3 查找下一个

Shift + F3 查找上一个

Alt + <- 光标上一个位置

Alt + -> 光标下一个位置

Ctrl + <- 上一个单词

Ctrl + -> 下一个单词

Ctrl + Shift + Space 触发参数提示

Ctrl + 鼠标 查看定义

Ctrl + Shift + O 符号跳转（查找函数等）

Ctrl + K + Ctrl + / 折叠注释

Ctrl + K + Ctrl + L 折叠切换：当前块

Ctrl + K + Ctrl + 数字0 全部折叠

Ctrl + K + Ctrl + J 全部展开

# 2.3 编辑类

Alt + Shift + F 格式化代码

Alt + J 【自定义（该快捷键为人为设置）】合并行

Ctrl + / 代码注释切换

Alt + 鼠标左键 多光标（可用于同时修改）

# 4 代码片段

# 4.1 构建代码片段

代码片段可以快速构建一段代码

"Print to console": { //Print to console 是代码片段的名称
    "prefix": "log",  //打出 log 后按 Tab 键，可以直接构造出 body 的内容
    "body": [
        "console.log('$1');", //$1 是占位符
        "$2"
    ],
    "description": "Log output to console"
}
